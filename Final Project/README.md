# 🎓 Course Management System

A modern Full Stack Course Management System built with **Angular**, **Node.js**, **Express.js**, and **MongoDB**. The application allows administrators to manage courses through a clean and responsive dashboard while providing secure user authentication.

## 🚀 Features

- User Authentication (Login & Register)
- JWT Authentication
- Responsive Dashboard
- Course Management (CRUD)
- Search Courses
- Course Details
- Protected Routes
- RESTful API
- Responsive UI
- Clean Architecture
- Error Handling
